## Overview
A web-based application for managing supervised driving experiences, including driver management, session tracking, and statistical analysis.

## Technologies
- PHP (PDO)
- MySQL
- HTML, CSS, JavaScript

## Technical Features
- Secure backend using **PDO prepared statements**
- Normalized **MySQL relational database**
- Full **CRUD operations** for drivers and sessions
- Automatic **AUTO_INCREMENT ID handling**
- Custom **driving statistics** calculations
- Dynamic frontend updates with JavaScript

## How to Run
1. Import the SQL file into phpMyAdmin  
2. Configure database credentials  
3. Run the project on a local web server
