Overview

A web-based application for managing supervised driving experiences, including driver management, session tracking, and statistics.

Technologies

PHP (PDO)

MySQL

HTML, CSS, JavaScript

Technical Features

Secure backend using PDO prepared statements

Normalized MySQL relational database

Full CRUD operations for drivers and sessions

Logical deactivation to hide inactive drivers

Optional permanent deletion of records

Automatic AUTO_INCREMENT ID management

Custom driving statistics calculations

Dynamic frontend updates using JavaScript

How to Run

Import the SQL file into phpMyAdmin

Configure database credentials

Run the project on a local web server
